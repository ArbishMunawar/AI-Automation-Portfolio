# 📄 AI-Powered Invoice Processing Automation

An automated invoice processing workflow built with **n8n**, **OpenAI**, and **Google Sheets**. This project extracts data from PDF invoices, uses AI to identify key information, and stores the results automatically in a spreadsheet.

## 🚀 Features

- Upload or receive PDF invoices automatically
- Extract text from PDF files
- Use AI to extract invoice details
- Store extracted data in Google Sheets
- Reduce manual data entry
- Improve accuracy and efficiency

## 🛠️ Tech Stack

- n8n
- OpenAI
- Google Sheets API
- JavaScript
- PDF Text Extraction

## 📊 Workflow

Trigger
   ↓
Download PDF
   ↓
Extract from PDF
   ↓
AI Agent
   ↓
Code Node
   ↓
Google Sheets
```

## 📋 Extracted Information

The workflow extracts:

- Invoice Number
- Company Name
- Invoice Date
- Due Date
- Total Amount
